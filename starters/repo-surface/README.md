# repo-surface

First live starter for `launch`.

This starter creates a clean repository surface with:
- route-first README
- `.gitignore`
- `src`, `tests`, and `docs` directories
- a verification script

## Run

```bash
bash bootstrap.sh /tmp/my-repo my-repo
````

## What it gives you

* a usable top-level repo surface immediately
* a non-junky starting layout
* a fast path into real work without waiting on setup drift

## Notes

This starter is intentionally narrow.
Its job is to get a repository into a clean public-ready starting state fast.
